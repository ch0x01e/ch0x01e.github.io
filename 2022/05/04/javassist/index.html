前言

Java中所有的类都被编译为class文件来运行，在编译完class文件之后，类不能再被显示修改，而Javassist就是用来处理编译后的class文件，它可以用来修改方法或者新增方法，并且不需要深入了解字节码，还可以生成一个新的类对象。

创建对象

首先要使用maven引入javassist包

然后创建一个Boy对象，这里的话创建的对象不能放到以java开始的包里面，所以这里我自己创建了一个jst的包然后把这个新对象放到这个包下

然后生成的Boy的class文件如下

ClassPool的主要方法：

  • getDefault() 获取ClassPool对象，因为ClassPool是单例模式
  • appendClassPath() 把目录添加到ClassPath
  • insertClassPath() 在ClassPath插入jar
  • get() 根据名称获取CtClass对象
  • toClass() 将CtClass转为Class 一旦被转换则不能修改
  • makeClass() 创建新的类或接口

CtClass主要方法：

  • addConstructor() 添加构造函数
  • addField() 添加字段
  • addInterface() 添加接口
  • addMethod() 添加方法
  • freeze() 冻结类使其不能被修改
  • defrost() 解冻使其能被修改
  • detach() 从ClassPool中删除类
  • toBytecode() 转字节码
  • toClass() 转Class对象
  • writeFile() 写入.class文件
  • setModifiers() 设置修饰符

CtMethod继承CtBehavior，需要关注的方法：

  1. insertBefore 在方法的起始位置插入代码
  2. insterAfter 在方法的所有 return 语句前插入代码
  3. insertAt 在指定的位置插入代码
  4. setBody 将方法的内容设置为要写入的代码，当方法被 abstract修饰时，该修饰符被移除
  5. make 创建一个新的方法

生成了class文件以后，我们主要用两种方法去进行调用，一是反射调用，二是加载class文件
