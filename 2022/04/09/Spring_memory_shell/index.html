SpringеҶ…еӯҳй©¬еҗҲйӣҶ

жҖ»еӯ—ж•°: 2.5k | йҳ…иҜ»ж—¶й•ҝ: 12еҲҶй’ҹ

еүҚиЁҖ

еүҚйқўеӯҰдәҶTomcatзҡ„Servletе’ҢFilterеҶ…еӯҳй©¬пјҢ з„¶еҗҺеӯҰдәҶеҮ еӨ©зҡ„springпјҢе®ЎдәҶдёҖдёӘзЁӢеәҸпјҢе°ұз»§з»ӯйЎәдҫҝжҠҠеҶ…еӯҳй©¬зҡ„дёңиҘҝеј„е®Ңеҗ§гҖӮеүҚйқўиҜҙеҲ°Servletе’ҢFilterеҶ…еӯҳй©¬е…¶е®һйғҪжҳҜеңЁй…ҚзҪ®ж–Үд»¶дёӯжіЁеҶҢдёҖдёӘServletжҲ–иҖ…FilterпјҢеӣ дёәд»–д»¬зҡ„з”ҹе‘Ҫе‘ЁжңҹжҳҜеҲ°жңҚеҠЎеҷЁз»“жқҹжүҚз»“жқҹгҖӮе…¶е®һSpringзҡ„controllerзҡ„еҶ…еӯҳй©¬д№ҹжҳҜеҰӮжӯӨпјҢд№ҹжҳҜжіЁеҶҢдёҖдёӘControllerжқҘе®һзҺ°гҖӮеӣ жӯӨжҲ‘д»¬йңҖиҰҒдәҶи§ЈSpringMVCдёӯControllerжіЁеҶҢзҡ„жөҒзЁӢгҖӮ

ControllerжіЁеҶҢжөҒзЁӢ

SpringMVCеҲқе§ӢеҢ–ж—¶пјҢеңЁbeanзҡ„жһ„йҖ ж–№жі•д»ҘеҸҠи®ҫзҪ®еұһжҖ§д№ӢеҗҺдјҡдҪҝз”ЁInitializingBean#afterPropertiesSetиҝӣиЎҢBeanзҡ„еҲқе§ӢеҢ–ж“ҚдҪңпјҢе®һзҺ°зұ» RequestMappingHandlerMapping з”ЁжқҘеӨ„зҗҶе…·жңү @Controller жіЁи§Јзұ»дёӯзҡ„ж–№жі•зә§еҲ«зҡ„ @RequestMapping д»ҘеҸҠ RequestMappingInfo е®һдҫӢзҡ„еҲӣе»әпјҢAbstractHandlerMappingжҳҜRequestMappingHandlerMappingзҡ„зҲ¶зұ»пјҢе№¶дё”AbstractHandlerMappingе®һзҺ°дәҶInitializingBeanжҺҘеҸЈпјҢжүҖд»Ҙе…ҲжқҘзңӢдёҖдёӢRequestMappingHandlerMapping#afterPropertiesSet

йҰ–е…ҲеҲқе§ӢеҢ–дәҶдәҶдёҖдёӘconfigеҜ№иұЎз»ҷиҮӘиә«зҡ„configеұһжҖ§пјҢе№¶дё”и®ҫзҪ®еҜ№еә”зҡ„еҖјпјҢз„¶еҗҺи°ғз”Ёзҡ„зҲ¶зұ»зҡ„afterPropertiesSetж–№жі•пјҢд№ҹе°ұжҳҜAbstractHandlerMapping#afterPropertiesSet

AbstractHandlerMapping#afterPropertiesSetи°ғз”ЁдәҶinitHandlerMethodsж–№жі•пјҢеҫӘзҺҜйҒҚеҺҶSpring дёӯжіЁеҶҢзҡ„ Beanе№¶дё”дҪҝз”ЁprocessCandidateBeanж–№жі•иҝӣиЎҢеӨ„зҗҶ

processCandidateBeanж–№жі•пјҢеҲӨж–ӯдәҶbeanTypeдёҚдёәз©әе№¶дё”жҳҜHandlerпјҢе°ұи°ғз”ЁdetectHandlerMethodsж–№жі•пјҢиҝҷйҮҢзҡ„isHandlerе°ұжҳҜеҲӨж–ӯbeanTypeжҳҜеҗҰжңүRequestMappingе’ҢControllerзҡ„жіЁи§Ј

йҰ–е…ҲйҖҡиҝҮеҸҚе°„иҺ·еҸ–еҪ“еүҚController Beanзҡ„еҜ№иұЎпјҢе№¶дё”иҺ·еҸ–еҪ“еүҚbeanзҡ„жүҖжңүHandler MethodпјҢж №жҚ®methodе®ҡд№үжҳҜеҗҰеёҰжңүRequestMappingжіЁи§ЈпјҢеҰӮжһңжңүпјҢж №жҚ®жіЁи§ЈеҲӣе»әRequestMappinginfoеҜ№иұЎ

return getMappingForMethod(method, userType);еҲӣе»әдәҶRequestMappinginfoеҜ№иұЎпјҢmethods.forEach((method, mapping)еӨ„еҫӘзҺҜйҒҚеҺҶmethodпјҢе№¶дё”йҖҡиҝҮregisterHandlerMethodжҠҠhandler methodдёҺеҜ№еә”зҡ„RequestMappingInfo еҲӣе»әжҳ е°„

registerHandlerMethodи°ғз”ЁmappingRegistry.registerж–№жі•

жҠҠе…ій”®дҝЎжҒҜеӯҳеӮЁеҲ°иҮӘиә«зҡ„mappingLookupпјҢurlLookupпјҢcorsLookupпјҢregistryеұһжҖ§дёӢ

зј–еҶҷеҶ…еӯҳй©¬зҡ„жӯҘйӘӨеҰӮдёӢ

  • еҲ©з”ЁspringеҶ…йғЁж–№жі•иҺ·еҸ–context
  • иҺ·еҫ— RequestMappingHandlerMapping зҡ„е®һдҫӢ
  • йҖҡиҝҮеҸҚе°„иҺ·еҫ—иҮӘе®ҡд№үControllerзҡ„жҒ¶ж„Ҹж–№жі•зҡ„MethodеҜ№иұЎ
  • е®ҡд№үеҜ№еә”зҡ„URLең°еқҖ
  • е®ҡд№үи®ҝй—®зҡ„ж–№жі•

Interceptor

еңЁSpring MVCдёӯInterceptorз”ЁжқҘжӢҰжҲӘе®ўжҲ·иҜ·жұӮпјҢиҝӣиЎҢеӨ„зҗҶжқҘе®һзҺ°жҹҗдәӣеҠҹиғҪ

еңЁWebеә”з”ЁејҖеҸ‘ж—¶пјҢйңҖиҰҒз»§жүҝSpring MVCзҡ„HandlerInterceptorAdapterжҠҪиұЎзұ»пјҢе®һзҺ°зҲ¶зұ»зҡ„жҠҪиұЎж–№жі•preHandler(),postHandler(),afterCompletion(),еңЁдёҖдёӘwebеә”з”ЁдёӯеҸҜд»Ҙе®һзҺ°еӨҡдёӘInterceptor,з»„жҲҗдёҖдёӘжӢҰжҲӘеҷЁй“ҫ,е…¶дёӯж–№жі•зҡ„жү§иЎҢйЎәеәҸеҰӮдёӢжүҖзӨә

е№¶дё”еңЁй…ҚзҪ®ж–Үд»¶дёӯжіЁеҶҢиҝҷдёӘInterceptor

еҗҜеҠЁжҲ‘д»¬зҡ„TomcatжңҚеҠЎеҷЁпјҢи®ҝй—®еҚіеҸҜжҲҗеҠҹе‘Ҫд»Өжү§иЎҢ

жҺҘдёӢжқҘе…·дҪ“еҲҶжһҗдёҖдёӢInterceptorзҡ„жіЁеҶҢжөҒзЁӢпјҢе…ҲеңЁTestInterceptorдёӯжү“дёӘж–ӯзӮ№пјҢејҖеҗҜи°ғиҜ•пјҢе…·дҪ“и°ғз”Ёж ҲеҰӮдёӢ

йҰ–е…ҲжҳҜеңЁHandlerExecutionChain#applyPreHandlerи°ғз”Ёзҡ„TestInterceptor#preHandlerпјҢе…ҲйҖҡиҝҮgetInterceptorsиҺ·еҸ–еҲ°дәҶжүҖжңүзҡ„InterceptorпјҢз„¶еҗҺеҜ№д»–иҝӣиЎҢйҒҚеҺҶпјҢи°ғз”Ёinterceptor.preHandleж–№жі•

з»§з»ӯеҫҖдёҠжүҫпјҢеңЁDispatcherServlet#doDispatchдёӯи°ғз”ЁдәҶHandlerExecutionChain#applyPreHandlerпјҢйҖҡиҝҮmappedHandler = getHandler(processedRequest);иҺ·еҸ–еҲ°дәҶдёҖдёӘHandlerExecutionChainеҜ№иұЎ

з„¶еҗҺеңЁжңҖеҗҺи°ғз”ЁmappedHandler#applyPreHandle,зҺ°еңЁжҲ‘д»¬еј„жё…жҘҡдәҶд»–е…·дҪ“жҖҺд№Ҳи°ғз”ЁпјҢдҪҶжҳҜжҲ‘д»¬еҰӮжһңиҰҒеҠЁжҖҒжіЁеҶҢInterceptorзҡ„иҜқиҝҳйңҖиҰҒзҹҘйҒ“д»–жҳҜд»Һе“ӘйҮҢиў«ж·»еҠ иҝӣжқҘпјҢжҲ‘д»¬еҸҜд»ҘзңӢеҲ°еңЁinterceptorsдёӯжңүжҲ‘д»¬жүӢеҠЁж·»еҠ зҡ„TestInterceptorеҜ№иұЎпјҢиҖҢinterceptorsжҳҜйҖҡиҝҮgetHandlerж–№жі•иҝӣиЎҢиҺ·еҸ–зҡ„пјҢжүҖд»ҘиҝҷйҮҢиҝӣеҲ°getHandlerж–№жі•зңӢзңӢ

йҒҚеҺҶthis.handlerMappingsпјҢиҺ·еҸ–HandlerMappingзҡ„е®һдҫӢпјҢеҶҚи°ғз”ЁgetHandlerж–№жі•пјҢз»§з»ӯи·ҹиҝӣgetHandlerж–№жі•пјҢиҝҷйҮҢзҡ„иҜқжҳҜи°ғз”Ёзҡ„mappingзҡ„getHandlerж–№жі•пјҢmappingеҸҲжҳҜthis.handlerMappingsдёӯзҡ„е…ғзҙ 

иҝҷйҮҢе®һйҷ…дёҠжҳҜи°ғз”Ёе®һйҷ…дёҠи°ғз”ЁдәҶorg.springframework.web.servlet.handler.AbstractHandlerMappingзұ»дёӯзҡ„getHandlerж–№жі•

и·ҹиҝӣgetHandlerExecutionChainж–№жі•

йҒҚеҺҶдәҶthis.adaptedInterceptorsпјҢе№¶еҲӨж–ӯnterceptorе®һдҫӢжҳҜдёҚжҳҜMappedInterceptorзұ»зҡ„е®һдҫӢеҜ№иұЎпјҢиҖҢMappedInterceptorзұ»е°ұжҳҜеҜ№жӢҰжҲӘеҷЁHandlerInterceptorжҺҘеҸЈзҡ„е®һзҺ°пјҢжүҖд»ҘеүҚйқўе®ҡд№үзҡ„TestInterceptorиҮӘз„¶дјҡиў«еҠ е…Ҙchainдёӯе№¶иҝ”еӣһпјҢжүҖд»ҘжҲ‘д»¬йңҖиҰҒеҠЁжҖҒжіЁеҶҢInterceptorеҸӘйңҖиҰҒеңЁorg.springframework.web.servlet.handler.AbstractHandlerMappingзұ»зҡ„е®һдҫӢеҜ№иұЎзҡ„adaptedInterceptorsж·»еҠ жҒ¶ж„Ҹinterceptorе®һдҫӢ

Interceptorе…·дҪ“е®һзҺ°

иҝҷйҮҢжҲ‘д№ҹжҳҜеҸӮиҖғдәҶеҲ«зҡ„еёҲеӮ…зҡ„ж–Үз« пјҢдҪҶжҳҜжҜҸдёӘеёҲеӮ…зҡ„й©¬еҘҪеғҸеҲ°жҲ‘иҝҷйғҪжІЎжі•з”ЁпјҢжүҖд»ҘжҲ‘иҝҷйҮҢж”№дәҶдёҖдёӢеҲ«зҡ„еёҲеӮ…д»¬зҡ„еҶ…еӯҳй©¬пјҢжҖ»дҪ“жқҘиҜҙеҲҶдёӢйқўеҮ жӯҘ
